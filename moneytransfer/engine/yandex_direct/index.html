Yandex Direct

Общее

Этот engine предоставляет доступ к финансовым операциям для работы с yandex direct Дока

payload

“engine” - используется для определения что мы будем использовать

“company” - идентификатор компании в системе для определения кред для подключения

“operation” - имя операции (описание ниже)

“method_params” - параметры запроса зависят от выбранной операции (описание ниже)

operation & method_params

  • “AccountManagement.Limits” - получение остатков по аккаунтам

 "accounts" - список аккаунтов по которым надо получить статитсику
  • “Campaign.CreditLimits” - получение остатка кредитного лимита

для данной операции ничего указывать не надо
  • “Campaign.PayCampaigns” - внесение денег на кампанию

'currency' - валюта зачисления

'amount' - сумма в валюте

'campaign' - идентификатор кампании
  • “Campaign.TransferMoney” - перевод между кампаниями

'currency' - валюта операции

'amount' - сумма в валюте

'campaign_from' - идентификатор кампании с которой осуществляется перевод

'campaigns_to' - список объектов на которые требуется перевести средства структура налаогична "Campaign.PayCampaigns"
  • “AccountManagement.Deposit” - внесение денег на аккаунт

'currency' - валюта операции

'amount' - сумма в валюте

'account': - идентификатор аккаунта
  • “AccountManagement.TransferMoney” - перевод денег между аккаунтами

'currency' - валюта операции

'amount' - сумма в валюте

'account' - идентификатор аккаунта куда

'account_from' - идентификатор аккаунта откуда

Примеры

Пример 1

Получение остатков по компании


payload = {
    'engine': 'yandex_direct',
    'company': 1,
    'operation': 'AccountManagement.Limits',
    'payload': {
        'accounts': [1, 2, 3]
    }
}

result = requests.post(URL, json.dumps(payload))
print(result.json())

Пример 2

Получение кредитного лимита


payload = {
    'engine': 'yandex_direct',
    'company': 1,
    'operation': 'Campaign.CreditLimits',
    'method_params': {}
}

result = requests.post(URL, json.dumps(payload))
print(result.json())

Пример 3

Зачисление на кампанию


payload = {
    'engine': 'yandex_direct',
    'company': 1,
    'operation': 'Campaign.PayCampaigns',
    'method_params': {
        'currency': 'RUB',
        'amount': 1000,
        'campaign': 1
    }
}

result = requests.post(URL, json.dumps(payload))
print(result.json())

Пример 4

Перевод с кампании на кампанию


payload = {
    'engine': 'yandex_direct',
    'company': 1,
    'operation': 'Campaign.TransferMoney',
    'method_params': {
        'currency': 'RUB',
        'amount': 1000,
        'campaign_from': 1,
        'campaigns_to': [
            {
            'currency': 'RUB',
            'amount': 500,
            'campaign_id': 2
            },
            {
            'currency': 'RUB',
            'amount': 500,
            'campaign_id': 3
            }
        ]
    }
}

result = requests.post(URL, json.dumps(payload))
print(result.json())
