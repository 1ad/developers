Yandex Direct

Общее

Этот “engine” предоставляет возможность создания аккаунта в Яндекс.Директ.

Документация Яндекса на метод add

payload

Параметры которые хранятся в method_params, передаются в Яндекс без изменений.

  • access - словарь для передачи параметров, по которым из базы будет получен access_token. Передать необходимо оба параметра: client_id и client_login.
  • Login - Логин пользователя Директа — главного представителя рекламодателя.
  • FirstName - Имя пользователя (не более 20 символов). Не допускаются символы &=<>.
  • LastName - Фамилия пользователя (не более 20 символов). Не допускаются символы &=<>.
  • Currency - Валюта рекламодателя.
  • header_client_login - Название аккаунта клиента ЯД. Этот параметр добавляется в headers как параметр Client-Login.
  • notification_email - Адрес электронной почты для отправки уведомлений, связанных с аккаунтом. В документации Яндекса соответствует Notification.Email.
  • notifications_list - Настройки SMS и email-уведомлений для главного представителя рекламодателя. Здесь списком перечисляются параметры из Notification.EmailSubscriptions.Option.
  • settings_list - Настройки рекламодателя. Здесь списком перечисляются параметры из Settings.Option (см. документацию Яндекса).
  • grants_list - Полномочия рекламодателя по управлению кампаниями. Здесь списком перечисляются параметры из Grants.Privilege (см. документацию Яндекса).

Пример

Чтобы повторить этот запрос через какое-либо приложение необходимо на API_HOST послать body отформатированный как JSON.

import json
import requests

API_HOST = "http://accmanager.apis.kb.1ad.ru/account_manager"

body = {"engine": "yandex_direct",
        "payload": {
            "method": "add",
            "access": {"client_id": "1234", 
                       "client_login": "test_client"},
            "method_params": {"Login":    "test_Login",
                              "FirstName":"test_FirstName",
                              "LastName": "test_LastName",
                              "Currency": "RUB"
                              },
            "params_modifier": {"header_client_login": None,
                                "notification_email": "test@realweb.ru",
                                "notifications_list": ["RECEIVE_RECOMMENDATIONS", 
                                                       "TRACK_POSITION_CHANGES"],
                                "settings_list": ["CORRECT_TYPOS_AUTOMATICALLY", 
                                                  "DISPLAY_STORE_RATING"],
                                "grants_list": ["EDIT_CAMPAIGNS", 
                                                "IMPORT_XLS", 
                                                "TRANSFER_MONEY"],
                                }
        }
        }

print(body)
res = requests.post(API_HOST,  data=json.dumps(body))
print(res.json())
