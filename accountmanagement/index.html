Пример обращения в API

В качестве базавого URL используется accountmanagement.apis.devision.io Для того чтобы отправить запрос необходимо отправить на выбраный вами метод http сообщение содержащие токен в заголовке Authorization:Bearer и запрос в формате JSON после параметра -d

curl -i -X POST \
   -H "Authorization:Bearer eyJ...arSQ" \
   -H "Content-Type:application/json" \
   -d '{ "client_id": "12345", "balance_slice": "ENGINE" }' \
   'http://accountmanagement.apis.devision.io/v1beta1/budget-order/limit'

Описание ошибок

При нормальной работе сервиса вы всегда будете получать ответ со статусом 200, код ошибки можно получить по ключу “code”. В нашем сервисе мы используни gRPC поэтому ошибки всегда будут соответсвовать этому документу https://cloud.google.com/inference/reference/rest/v1/Code?hl=ru
