Ресурсы

BudgetOrder

Управление бюджетом в рекламных системах

Информация о текущем бюджете

POST /v1beta1/budget-order/get
Описание

Получение информации о текущем состоянии бюджета на аккаунтах клиента и общем кредитном лимите

Параметры
Тип Имя Схема
Body body
обязательно		 BudgetOrderGetRequest
Ответы
HTTP код Схема
200 BudgetOrderGetResponse

Изменение бюджета

POST /v1beta1/budget-order/mutate
Описание

Изменять можно в большую или меньшую сторону

Параметры
Тип Имя Схема
Body body
обязательно		 BudgetOrderMutateRequest
Ответы
HTTP код Схема
200 BudgetOrderMutateResponse

Определения

BudgetOrderGetRequest

Имя Описание Схема

BudgetOrderGetResponse

Имя Описание Схема

необязательно		 остаток на кредитной линии number (float)

Имя Описание Схема

обязательно		 Сумма для изменения, положительная при увеличения бюджета и отрицательная при уменьшения number (float)

OrderScope

Имя Описание Схема
client_id
обязательно		 Devision Client Id
Клиент в рамках которого производится операция		 string (int64)
agency_login
обязательно		 Логин (CustomerId) агентства с кредитной линией string
operation_id
необязательно		 Обязательный для операция, изменяющих данные. Уникальный id операции.

можно просто сделать повторный запрос с этим id и получить ответ		 string
