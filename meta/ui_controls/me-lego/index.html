Элемент me-lego

Основной элемент-контейнер для рендеринга элементов. Рендерится в me-page как основной контейнер, а так же может быть использован внутри себя и себе подобных контейнеров: me-lego, me-lego-list, me-wizard

attrs [object Object]

defaultValue (any)

Значение элемента по умолчанию Для некоторых элементов

depends (string)

Зависимости от других элементов. При изменении зависимых элементов, при условии, что заполнены все значения из depends делается автоматический переапрос refPvmData и depends будут переданы в env.sp как {key: value} **@example** https://meta.realweb.ru/page?p=4329&a=35 **@example** https://meta.realweb.ru/page?p=3597&a=35

elems [object Object]

entityId (number)

Meta EntityId Для некоторых элементов, напрмиер [[IMeInput]], [[IMeTable]]

formHorizontal (boolean)

Если элемента поддерживает, вывести его в стиле формы ключ, значение. Чтобы заголовок был на уровне элемента

help (string)

Доп информация, рисуется под label или под элментом Зависит от типа элемента

icon (string)

Иконка Font Awesome Например fa-bicycle

@link http://ui.adplatform.lan/Static_Full_Version/icons.html

id (string)

ID данных элемента Будет доступно в env.sp.{ID} или как вложенная структура, если это [[IMeElemContainer]]

label (string)

Заголовок элемента Обычно над элементом, на некоторых не рисуется вообще

name (string)

Название элемента Например: me-lego, me-input

offset (number)

Отстут по сетке справа от 0 до12

refPvmData

Инструкции по получению данных для [[IMeInputAttrs.options]] или таблицы

@example https://meta.realweb.ru/page?p=3570&a=35

@example https://meta.realweb.ru/page?p=3597&a=35

refPvmValidator

Пользовательская валидация

@example https://meta.realweb.ru/page?p=3359&a=30

selfDepends (boolean)

Зависимость от себя самого, нужно для editable [[IMeTable]] и прототип для поиска [[IMeInputAttrs.options]] в me-input для частичной подгрузки данных

@beta

span (number)

Размер по сетке справа от 0 до12

value (string)

для некоторых элементов
